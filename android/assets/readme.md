# Assets

Place the following image assets in this folder before building:

- `icon.png` — app icon (1024x1024)
- `splash.png` — splash screen (1242x2436)
- `adaptive-icon.png` — Android adaptive icon foreground (1024x1024)
- `favicon.png` — web favicon

These are referenced by `app.config.ts`. The build will fail without them.
